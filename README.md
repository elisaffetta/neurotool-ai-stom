## Neurotool AI — SEO-тексты для стоматологии

Этот репозиторий содержит фронтенд‑лендинг для сервиса **SEO‑текстов для стоматологических клиник**.
Страница описывает продукт Neurotool AI, тарифы, процесс работы и преимущества, а также включает калькулятор стоимости.

---

## Технологии

- **React 18**
- **TypeScript**
- **Vite** (dev‑сервер и сборка)
- **React Router** (маршрутизация)
- **shadcn-ui + Radix UI** (UI‑компоненты)
- **Tailwind CSS** (стили)
- **TanStack Query** (инфраструктура для работы с данными)

---

## Запуск проекта локально

Требуется установленный **Node.js** (желательно через nvm) и **npm**.

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера (по умолчанию порт 8080)
npm run dev
```

После запуска приложение будет доступно по адресу:

- http://localhost:8080

---

## Сборка и предпросмотр

```bash
# Продакшен-сборка
npm run build

# Предпросмотр собранной версии
npm run preview
```

## Структура проекта (кратко)

- `src/App.tsx` — корневой компонент приложения, настройка роутинга и провайдеров.
- `src/pages/Index.tsx` — главная страница‑лендинг.
- `src/components/*` — секции лендинга и UI‑компоненты.
- `index.html` — HTML‑шаблон и SEO‑мета‑теги.
